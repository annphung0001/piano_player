Ứng dụng giải trí PIANO PLAYER
Nhóm thực hiện: Nhóm 20 DSA
Để chạy ứng dụng, chạy file "PianoPlayer.java"
Để hiểu các chức năng hãy bấm nút ? trong chương trình.

File hpbd.txt chứa thông tin bài hát Happy Birthday.
File Saved.txt chứa thông tin bài hát được ghi lại.

Phương thức scanIn() nằm trong HappBD.java để chuyển file.txt thành Queue.
